# 🍔 Food Ordering System

A simple and user-friendly **Food Ordering System** developed using **PHP** and **MySQL**. This project allows users to manage restaurant orders through complete CRUD (Create, Read, Update, Delete) operations.

## 📌 Project Overview

This project was developed as a semester lab assignment to practice web development concepts. It helps restaurant staff manage customer orders digitally instead of using paper records.

## ✨ Features

- ➕ Add New Orders
- 📋 View All Orders
- ✏️ Update Existing Orders
- 🗑️ Delete Orders
- 🔍 Search Orders

## 🚀 How to Run

1. Install XAMPP.
2. Start Apache and MySQL.
3. Copy the project folder into the `htdocs` directory.
4. Create a database named `food_ordering_system`.
5. Import the SQL file or create the `orders` table.
6. Open your browser and visit:

```
http://localhost/food_ordering_system
```
